Q1. Orthogonal Matching Pursuit (OMP) is a greedy sparse recovery method. Starting from the full residual equal to the signal, it computes correlations between the residual and all atoms in the dictionary. It picks the atom with the largest absolute correlation, appends that atom index to the support set, and then solves a least-squares problem restricted to the chosen support. Using that least-squares estimate, it reconstructs the signal contribution and updates the residual. The algorithm repeats this pick-refit-update cycle until either the desired sparsity level is reached or the residual norm becomes small.

Q2. One way to accelerate OMP is to use Stagewise OMP (StOMP). Instead of adding only one atom per iteration, it adds multiple atoms whose correlations exceed a threshold. This reduces the number of outer iterations. The logic is that when several atoms are clearly relevant, committing to all of them at once can save time while still allowing a later least-squares refit.
