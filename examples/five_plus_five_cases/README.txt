This zip contains 10 synthetic submission files and a grading script.

Structure:
- submissions/A : 5 OMP-question submissions
- submissions/B : 5 sociology-question submissions
- sources/internet_like : reference texts used to simulate external plagiarism
- scripts/grade_cases.py : grades all 10 submissions and writes results.json and results.md

Usage:
1. Unzip
2. Run:
   python scripts/grade_cases.py

Outputs:
- results.json
- results.md
